# Excel--Vrinda-Store-Data-Analysis
Vrinda Store Data Analysis Dashboard | Microsoft Excel

Overview:
This project is an interactive sales analysis dashboard created using Microsoft Excel to analyze and visualize Vrinda Store’s annual sales performance. The dashboard helps track sales trends, customer behavior, order status, channel performance, gender-wise sales, and top-performing states through dynamic charts and filters.

The project transforms raw sales data into meaningful business insights, enabling better understanding of customer preferences and business performance.

# Purpose of the Project:
The main purpose of this project is to analyze Vrinda Store’s sales data and transform raw information into meaningful business insights using Microsoft Excel. The dashboard helps track sales performance, customer behavior, order trends, and channel-wise performance to support data-driven decision-making and business growth.

1.Key Features
* Interactive dashboard with slicers and filters
* Orders vs Sales analysis
* Sales analysis by gender and age group
* Order status tracking
* Channel-wise sales performance
* Top 5 state-wise sales analysis
* Dynamic charts and business insights generation

2.Tools & Technologies Used
* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning & Formatting
* Dashboard Design
* Data Visualization

3.Dashboard Insights
* Women customers contributed higher sales compared to men
* Amazon and Myntra were the top-performing sales channels
* Maharashtra generated the highest sales among all states
* Majority of orders were successfully delivered
* Adult age group contributed the highest number of purchases

4.Business Impact

This dashboard helps businesses monitor sales performance, understand customer trends, optimize sales channels, and make informed business decisions through visual analytics and interactive reporting.
